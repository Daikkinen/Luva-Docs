Política de Privacidad Última actualización: 19 de junio de 2026 El bot de Luva es una herramienta privada utilizada exclusivamente para la administración del servidor oficial de Discord de Luva. Datos recopilados El bot puede acceder a información proporcionada por Discord necesaria para su funcionamiento, incluyendo: * ID de usuario. * ID de servidor. * ID de canales. * Roles de usuarios. * Contenido de mensajes cuando sea necesario para ejecutar comandos o funciones de moderación. Uso de los datos Los datos se utilizan únicamente para: * Verificación de usuarios. * Asignación de roles. * Moderación. * Administración del servidor. Almacenamiento de datos No se venden ni comparten datos con terceros. Inteligencia Artificial Los datos obtenidos por el bot no se utilizan para entrenar modelos de inteligencia artificial o aprendizaje automático. Contacto Para cualquier consulta relacionada con esta política, los usuarios pueden contactar al equipo administrativo del servidor de Luva.
